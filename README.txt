== Description ==

RevealFX for Elementor is a powerful addon that enhances your Elementor widgets with stunning reveal animations. With easy-to-use controls and customizable parameters, you can create engaging and dynamic effects that improve user experience and visual appeal.

Key Features:
Seamless integration with Elementor.
Multiple animation styles and effects.
Customizable duration, delay, and easing functions.
Lightweight and optimized for performance.
No coding required!

Transform the way elements appear on your pages and add professional-level animations with just a few clicks!

== Installation ==

1. Download the plugin ZIP file.
2. Upload it to your WordPress site via Plugins > Add New > Upload Plugin.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to Elementor and start using the RevealFX options inside any widget's advanced settings.

== Frequently Asked Questions ==

= Does this plugin work with all Elementor widgets? =
Yes! RevealFX is designed to work seamlessly with all Elementor widgets and sections.

= Will it slow down my website? =
No! RevealFX is lightweight and optimized to ensure minimal impact on performance.

= Can I use it with Elementor Pro? =
Absolutely! It works with both the free and Pro versions of Elementor.
